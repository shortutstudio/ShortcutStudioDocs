Native connectors for Shortcuts

Connect services to Apple Shortcuts.

ShortcutStudio links third‑party services using secure OAuth, stores tokens in Keychain, and surfaces last sync and failure causes so you always know what happened.

Privacy‑minded

Tokens are stored in the system Keychain; sign-in uses ASWebAuthenticationSession (no embedded webviews).

Transparent status

Shows last successful sync and last failure cause/time to help you fix issues quickly.

Disconnect anytime

Disconnect removes tokens locally and attempts provider revocation when supported.
