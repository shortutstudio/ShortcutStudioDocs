# ShortcutStudio Website

This repository hosts the ShortcutStudio website and policy pages.

## Pages

- Home: `/`
- Privacy Policy: `/privacy/`
- Support: `/support/`

## Deployment

The site is intended to be published via GitHub Pages.

In GitHub: **Settings → Pages → Build and deployment**
- Source: **Deploy from a branch**
- Branch: `main`
- Folder: `/ (root)`
