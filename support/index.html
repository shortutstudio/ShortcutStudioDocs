Common issues

  • Sign-in failed: confirm you’re using OAuth via the system browser prompt and try “Fix connection”.
  • Auth expired: reconnect the service in the app; the app will show the last failure cause/time.
  • Offline: retry when online.

Disconnect / revoke

You can disconnect a service in the app at any time. This removes tokens from Keychain and attempts to revoke provider access when supported.
